Case Study: User flow

Project Overview

This project is a donut app that I designed using Figma. The prototype features an interactive carousel that users can hold and drag to view other donuts for sale. The goal of my design is to make the user flow smooth and easy to navigate and as well as endengaging and interactive.

Problem Statement

The problem presented was to simply create a user flow for the process of finding products, viewing its details, adding it to cart and checking out.

Design Process

To start, I did some research on other apps to get some ideas on what I could incorporate into my design as well as take notes on any pain points. Firstly, I developed a journey map starting from the homepage to selecting a product, its details page, adding the product to cart and finally, the checkout page. From then, I sketched a couple of low-fidelity wireframes of each page in order to tackle the problem. After reviewing each wireframe, I picked out the solution that I felt allowed for a user friendly experience. After that, I went into Figma to start creating the high-fidelity wireframes.
