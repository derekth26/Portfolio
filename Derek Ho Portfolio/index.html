About Me

I'm currently in my last semester in the DMIT Web Design and Development program at NAIT and will graduate in December. I love the design and UX/UI aspects of web development and enjoy the entire process—from storyboarding and mood boards to wireframes and bringing user-friendly websites to life. I'm eager to apply the skills and knowledge I've gained at my time at NAIT to contribute to the web industry.

My Projects

TRVL

The TRVL website is a project that I made using React, Vite, Node js and used MUI and CSS to make the website

more interactive. The website is essentially is a travel app that allows user to browse tours that are available all around the world.

Goatkicks

GoatKicks is a fun side project I built to dive into e-commerce. I set up the site using WordPress and

used the WooCommerce plugin to handle all the shop functionalities. It’s a shoe store where visitors can browse various footwear options, check out product details, and make purchases.

ChefAnt Landing Page

The ChefAnt website was developed as part of a class project, initially intended to be a high-fidelity

wireframe using storyboarding and Figma. The project's goal was to create a compelling landing page for a chef who uploads and shares recipes, while also allowing users to submit their own recipes for publication. To bring this concept to life, I went beyond the wireframe and created a fully functional website using HTML, CSS, and JavaScript. The website features a clean and engaging interface designed to showcase the chef's recipes and facilitate user submissions.

FetchPokédex

I built a Pokédex site where users can search through thousands of Pokémon. The main functionality of

the site is powered by JavaScript, leveraging the Pokémon API to retrieve data in real-time, such as Pokémon names, types, stats, and abilities. In addition, the layout and design of the Pokédex were created using CSS.

ToDoList

The ToDoList was created using React, Vite, Nodejs, MUI and CSS. This website allows users to

add and delete tasks. It also allows users to reorganize their list of tasks by moving them up or down.

Planet Catalogue

For one of my classes, I built a planet catalogue site using Bootstrap, PHP, and MySQL. The project

required incorporating CRUD functionalities, allowing users to view, search, add, and edit planets. To manage entries, users need to log in with the following credentials: username: yourname, password: yourpassword. While the design of the site prioritizes functionality over aesthetics, as the assignment's focus was on implementing these features, it's functional and serves its purpose.

High-fidelity Wireframes
